Проект по изучению языка програмирования Python и луших практик :-)

evecli.g_id(name)[исходный код]¶

Функция. Возвращает id товара.

Параметры:name (str) – Название региона
Результат:Данные с id товара
evecli.goods = ['Tritanium', 'Pyerite', 'Mexallon', 'Isogen', 'Nocxium', 'Oxygen']¶

list – Список товаров.

evecli.orders(region, good)[исходный код]¶

Заказы из заданных регионов, заданных товаров сохраняются в БД для дальнейшего использования.

Параметры:
  • region (str) – Region id.
  • good (str) – Goods id.
evecli.r_id(name)[исходный код]¶

Функция. Возвращает id региона.

Параметры:name (str) – Название региона
Результат:id региона
evecli.regions = ['Metropolis', 'Heimatar', 'Molden Heath', 'Derelik', 'Curse']¶

list – Список регионов.

evecli.sql_goods(name)[исходный код]¶

Функция поиска inventory_type в ESI API. Вызывается внутренняя функция g_id(name) которая возвращает id. Полученные данные сохраняются в БД. Если запись существует - результат игнорируется.

Параметры:name (str) – Название товара
evecli.sql_region(name)[исходный код]¶

Функция поиска id региона в ESI API. Вызывается внутренняя функция r_id(name) которая возвращает id. Полученные данные сохраняются в БД. Если запись существует - результат игнорируется.

Параметры:name (str) – Название региона
